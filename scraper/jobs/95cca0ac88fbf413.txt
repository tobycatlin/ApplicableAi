Frontend Developer - JavaScript, Typescript, React - Cambridge 
Frontend Engineer required to join a pioneering and cutting-edge Defense and Space manufacturing company, headquarted in Cambridge but with a global presence. 
About the Role: As a Front End Software Engineer you will play a crucial role in delivering high-quality software products and involved in developing frontend web UIs driven by APIs using technologies such as Javascript, Typescript, React, HTML, and CSS. . 
Front end Engineer Key Responsibilities: 
Develop high-quality application software through the entire software development life cycle 
Create functional designs, specifications, and costings 
Collaborate to deliver high quality software applications. 
Support the creation of user documentation as needed. 
Requirements 
Proficiency in creating and maintaining API-driven frontend web UIs using Javascript, Typescript, React, HTML, and CSS. 
Familiarity with configuring frontend build systems like WebPack, Rollup, esbuild, etc. 
Knowledge of containerization technologies like Docker. 
Ability to create and maintain high-performance networked software using an appropriate backend language (Python). 
At least 3 years of experience in a similar development role. 
Offer: Up to £85k Dependent on Experience + Excellent Benefits
